# 🎮 Tic Tac Toe2.0 🤖
LINK:-https://tic-tae-toe-game2-0.vercel.app/


An unbeatable Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

This version includes a smart AI powered by the **Minimax Algorithm**, making the computer impossible to beat.

---

## 🚀 Features

- ✅ Play vs Computer AI
- 🧠 Minimax Algorithm (Unbeatable AI)
- 🎨 Modern Responsive UI
- 🏆 Winner Highlight
- 🔄 Restart Button
- 💻 Clean and Structured Code

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Modern UI Design)
- JavaScript (Game Logic + AI)
